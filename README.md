## Overview
This project analyzes fraud patterns across transaction data to identify high-risk behaviors, peak fraud periods, and geographic concentrations using interactive dashboards.

---

## Business Impact & Key Insights

- ~20% of all transactions (~213K out of 1M) are classified as high-risk → indicates significant fraud exposure and the need for stricter detection rules  
- Fraud activity peaks at 16:00 (~17K transactions) → highlights critical monitoring windows for fraud prevention teams  
- Fraud is highly concentrated in specific regions such as California (~114K) and Texas (~81K) → supports targeted, region-based risk controls  
- Swipe transactions (~0.93M) significantly exceed online transactions (~0.11M) → suggests higher exposure in physical transaction channels  

---

## Value Delivered

- Enables data-driven fraud detection strategies  
- Helps prioritize monitoring during high-risk time windows  
- Supports regional risk segmentation for better control policies  
- Improves visibility into transaction behavior and risk distribution  

---

## Dashboard Features

- Fraud activity by hour  
- Transaction risk vs amount analysis  
- Geographic fraud distribution  
- Risk level segmentation  
- Top cities by fraud transactions  

---

## Tools Used

- Excel (Data Cleaning & Preprocessing)  
- Power BI (Data Modeling & Visualization)
